## Project Overview
The objective of this project was to analyze 48,000+ rows of pizza sales data to help a fictional restaurant optimize its inventory and staffing.

## Technologies Used

Python (Core language)

Pandas (Data manipulation)

Matplotlib & Seaborn (Data visualization)

Jupyter Notebook (Environment)

## Key Insights

Peak Hours: Dinner time (5:00 PM – 7:00 PM) is the busiest, followed by a lunch rush at 12:00 PM.

Busiest Days: Fridays and Saturdays generate the most revenue.

Seasonal Trends: Sales peaked in June (Summer surge) and dipped in October.
